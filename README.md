# UK Productivity Analysis

Independent consulting project analysing labour productivity 
trends across five major economies — UK, USA, Germany, 
France, and Japan — from 2000 to 2023.

## Data Source
World Bank World Development Indicators, accessed via 
the WDI package in R.

## Key Finding
The UK has experienced significant productivity stagnation 
relative to peer economies since 2008, with GDP per person 
employed remaining substantially below US levels and broadly 
below French and German levels by 2023.

## Tools Used
R, ggplot2, dplyr, WDI package

## How To Run
Open analysis.R in RStudio.
Run install.packages(c("tidyverse","WDI")) in console.
Run the script line by line.
